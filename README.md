# 📄 Professional Resume Builder

A premium, feature-rich web application designed to help professionals create stunning, industry-standard resumes in minutes. Built with a focus on aesthetics and user experience, this tool offers 35+ meticulously crafted templates and modern features to make your job application stand out.

![Resume Builder Preview](https://via.placeholder.com/800x400?text=Resume+Builder+Interface)

## ✨ Key Features

- **35+ Professional Templates**: Choose from a wide variety of designs ranging from Corporate and Creative to Minimalist and Elegant.
- **AI-Enhanced Content**: Built-in logic to help improve career objectives and suggest skills (requires backend API).
- **Draggable Sections**: Easily reorganize your resume sections (Education, Experience, Skills, etc.) with a simple drag-and-drop interface.
- **Real-Time Preview**: See exactly how your resume looks as you type.
- **High-Quality PDF Export**: Download your resume as a clean, professionally formatted PDF using `jsPDF` and `html2canvas`.
- **Photo Support**: Upload and crop your profile picture directly within the app.
- **Dark Mode**: A sleek dark interface for comfortable editing at any time.
- **Persistent Storage**: Your data is automatically saved to your browser, so you never lose your progress.

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Khangulamgousamjat/resume-builder.git
   ```
2. Open `index.html` in your browser.

## 🛠️ Built With

- **HTML5 & CSS3**: Modern styling with Glassmorphism and CSS variables.
- **JavaScript (Vanilla)**: Clean, efficient logic for state management and PDF generation.
- **html2canvas**: For capturing resume layouts.
- **jsPDF**: For generating professional PDF documents.
